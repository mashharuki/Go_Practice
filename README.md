# Go_Practice

Go 言語を学習するための練習用リポジトリです。

## 実行コマンド

```bash
go run ソース名.go
```

## ビルドコマンド

```bash
go build ソース名.go
```

## 実行コマンド

```bash
./ソース名
```

## はじめ方

```bash
go mod init MODULE_NAME
```

## fyne のデモアプリ動かし方

インストール

```bash
go get fyne.io/fyne/v2/...
```

```bash
go run fyne.io/fyne/v2/cmd/fyne_demo@latest
```
